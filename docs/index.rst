Installation
^^^^^^^^^^^^
VESIcal can be installed with pip:

.. code-block:: python

    pip install VESIcal

Always use the most up-to-date version of the code:

.. code-block:: python

   pip install VESIcal --upgrade

See the installation section for detailed instructions and dependencies.
